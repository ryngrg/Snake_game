# Snake_game
The snake game implemented in C using curses.h

### objective:<br>
* Eat cherries, depicted as 'X'
* Don't run snake over itself
* Don't go out of the frame

### controls:<br>
* move snake using arrow keys.<br>
* press 'q' to exit anytime
